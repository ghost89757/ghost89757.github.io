3D UX-Net：用于医学图像分割的大核体卷积现代化分层Transformer

一 引言

  • 大多数医学图像如MRI和CT都属于volumetric data（体积数据），为了更加充分利用体素信息，近几年提出了不少3D卷积模型，也取得一些进展。

  • 近几年，随着Transformer的横空出世，其惊人的效果不仅让其在NLP领域火爆，Vision Transformer的提出也让Transformer在图像领域大展拳脚，Convnet在图像领域的地位受到挑战。于是有人开始反击，想证明Convnet可以做出Transformer的效果，于是ConvNext对标SwinTransformer进行了实验，取得了不错的效果，这样的设计思路也启发了3D UX-Net的作者。

  • 3D UXNet时一种轻量级卷积神经网络，其使用ConvNet模块调整分层Transformer以实现稳健的体素分割，在三个具有挑战性的脑体积和腹部成像公共数据集取得了很好的效果。

三 设计思想

编码器模块的设计思想分为两个层面：1.block-wise（微观）2.layer-wise（宏观）

1.block-wise（微观）

  • patch-wise Features Projection（特征快的投影）：ConvNets和ViTs都是使用一个共同的基础模块将特征缩小到一定尺寸，而不是将图像块展平为具有线性层的顺序输入，因此作者采用了具有大卷积核的投影层来提取patch-wise特征作为编码器的输入。

  • volumetric depth-wise convolution with LKs（具有大卷积核的体深度卷积）：在Transformer中，使用窗口W-MSA和SW-MSA生成了跨层的感受野，并且进一步细化了非重叠窗口之间的特征对应关系。受深度卷积思想的启发，作者发现自注意力机制中的加权求和方法与每通道卷积基础之间的相似之处，使用LK大小的深度卷积可以在提取类似于MSA块的特征时提供大的感受野，并使的输入特征和输出特征具备相同的通道维度。

  • inverted bottleneck with depthwise convolutional scaling（具有深度卷积缩放的反向瓶颈结构）：Swin Trasformer的另一个结构，线性层的隐藏层维度比输入维度宽四倍。这与ResNet中的瓶颈结构类似，我们借鉴该结构，并将卷积层移到最上层。

2.layer-wise（宏观）

  • applying residual connections（加入残差链接）：如上如所示，这使四种网络模块的结构。图a为3D U-Net模块，用了两个3D卷积，每次卷积接BN和ReLU；图b为残差模块，是一个瓶颈结构；图c是Swin Transformer模块，用来一个窗口自注意力接全连接层；图d是本文的3d UX-Net模块，是一个具有深度卷积的瓶颈结构，其卷积模块进行了上移。

  • adapting layer normalizetion（采用层归一化）

  • using gelu as the activation layer（采用GeLU作为激活函数）

四 基础知识

1.从2DConvolution到3D Convolution

1.1 2D单通道卷积

如图，这是一个2D单通道卷积，我们重点看卷积的维度。

这里的输入图像有两个维度：高度h，宽度w；卷积核大小也有两个维度：高度h，宽度w；最终输出图像是二维的，也有两个维度：高度h，宽度w。

1.2 2D多通道卷积

如图，这是一个2D多通道卷积。输入图像有三个维度：高度h，宽度w，通道数c；卷积核也有三个维度：高度k，宽度k，通道数c；最终输出图像是二维的，有两个维度：高度h，宽度w。

这里我们可以把输入图像看作一个高度为h，宽为w，长为c的立方体，卷积核看作一个高度为k，宽度为k，长度为c的立方体。卷积核立方体再输入图像的立方体上面做二维的滑动运动，输出的是一个二维的图像。

1.3 3D单通道卷积

无论是视频还是3D图像，实际上都是对二维图像的堆叠，来达到现实中三维的效果。

假设我们这里输入的是一个视频，那么有层数为红、绿、蓝三层可以理解为我们的视频图像是RGB图像，而每个颜色层有也有三个图片可以理解为是三个RGB图像在每个颜色通道上进行了堆叠，所以输入的视频是三张RGB图像的堆叠。

我们仍旧认为输入图像有三个维度：高度h，宽度w，深度l（这里的l也可以表示时间）；卷积核也有三个维度：高度h，宽度w，深度d（d < l）；输出的图像是个二维的，有三个维度：高度h，宽度w，深度l（这个l和输入图像的l不等）。

三维卷积也把输入看作是一个立方体，卷积核看作是一个立方体，这和2D多通道卷积看起来很像，但是输出的是一个三维结果，这是为什么呢？

原因在于，三维卷积中的卷积核深度d < l，这使得卷积核不仅能在高和宽的二维方向做滑动运动，也可以在深度第三个维度上做滑动运动。

可以这么认为，输出结果的维度取决于卷积核滑动运动的维度。

1.4 3D多通道卷积

当我们输入的是一个三维RGB图像时，我们会发现，输入图像的维度有四个：高度h，宽度w，深度l，通道数c。那么我们的多通道3D卷积如何运算？

如图所示，由于有三个通道的三维图像，所以我们用一个k * k * k的卷积核分别对这三个三维图像进行卷积的时候，我们也会得到三个三维特诊图像，但是我们不这样做，而是将每个通道的卷积核卷积后的结果融合到一个输出结果，那么我们得到的输出图像就是一个融合了三个通道特征的三维图像。

1.5 3D卷积的运算

我们已经只管感受过了3D卷积，具体运算可以参考下图：

输入的视频是四张图片在时间顺序上的堆叠，我们按时间顺序对堆叠的四张图片依次命名为p1，p2，p3，p4；卷积核其实是两个二维卷积核的堆叠，按深度对其命名为k1，k2；我们以深度记录卷积，。运算过程如下：

  • 第一次卷积（黑色线）：k1在p1上做二维卷积，k2在p2上做二维卷积，乘积求和得到第一个二维的输出图片。
  • 第二次卷积（红色线）：k1在p2上做二维卷积，k2在p3上做二维卷积，乘积求和得到第二个二维的输出图片。
  • 第三次卷积（蓝色线）：k1在p3上做二维卷积，k2在p4上做二维卷积，乘积求和的得到第三个二维的输出图片。

可以看出，三维卷积核是现在高度h和宽度w的二维维度上滑动，然后再在深度l的维度上滑动，然后在进行二维滑动……每次二维滑动卷积输出的二维图片在一维滑动中堆叠，最终形成三维结果。这样，也很好的捕捉到了视频的动作特征。

3.残差网络（ResNet）

  • 神经网络的退化：深度神经网络通过增加网络的层数，Loss逐渐减小，然后趋于稳定达到饱和，然后再继续增加网络层数，Loss反而增大。

由神经网络的退化我们知道，神经网络层数不是越大越好的，而是由固定的层数，超过该层数，神将网络退化。在具体的训练网络时，我们不知道最优的网络层数是多少，所以一般网络层数会比较大，在优化是，网络中就多了一部分层是多余的，对于这些多余的层，我们希望它们的输入和输出一样，即恒等映射，以达到最优层数。这时，就需要引入残差快。

如图所示，这就是一个残差块的结构，其中x经过第一层权重，再经过激活函数，输出了F(x)，然后进入第二层权重之后，经过激活函数之前，我们让F(x) + x，这样经过激活函数之后，我们的输出H(x)就变为了H(x) = F(x) + x，我们的目的是恒等映射，所以只需要让F(x) = 0就行了，这时输出就是H(x) = x。

这个设计十分巧妙，要使输入输入等于输出，我们有两种方法，一种使通过改变权重，使得输入输出相同，但是我们知道训练权重的过程很复杂，不是我们很简单就能改变的；相比之下，第二种方法就显得十分巧妙，加入残差快使F(x) = 0，我们只需要给权重赋很小的值，就可以实现输出接近0。

4.瓶颈结构（Bottleneck Architecture）

瓶颈的特点是一头大，一头小，瓶颈结构主要针对网络结构中输入和输出的维度问题，常见于残差块中（ResNet Block）。

使用1 * 1的网络结构很方便改变维度，并且减少了参数，降低了计算量，提升了网络的灵活性。

5.激活函数（GELU）

我们采用GELU作为激活函数，其即具有RelU函数的优点，可以防止过拟合，也有Dorout的优点，有一定的泛化能力。

首先需要说明的是，ReLU和Dropout都会返回一个神经元的输出，其中，ReLU会确定性的将输入乘上一个0或者1，Dropout则是随机乘上0。而GELU也是通过将输入乘上0或1来实现这个功能，但是输入是乘以0还是1，是在同时取决于输入自身分布的情况下随机选择的。换句话说，是0还是1取决于当前的输入有多大的概率大于其余的输入。

这是3D U-Net的结构。

如图，这是3D UX-Net的整体结构，其与3D U-Net的结构很相似。

编码部分包括:

  • Large Kernel Block（大卷积核投影）
  • 4个Stage（阶段）
    • 每个Stage由一个3D UX-Net Block和一个Downsampling Block组成
      • 一个3D UX-Net Block由两个瓶颈结构（深度卷积层上移）组成
      • Downsamping用2 * 2 * 2的Kernel以Stride为2来进行卷积，将分辨率降低2倍，同时特征通道增加2倍。

解码部分包括：

  • 残差拼接：用两个”norm + 3 * 3 * 3 Conv”的残差块来提取特征
  • 用转置矩阵上采样

最终，我们将patches的残差特征，与上采样的残差特征拼接，经过一个带有带有Softmax函数的1 * 1 * 1卷积层来预测分割概率。

1.3 Dowmsampling Block（下采样部分）

下采样部分使用 2 * 2 * 2的卷积核，步长为 2 的卷积来进行下采样，使得分辨率降低为原来的一半，特征通道提高为原来的2倍。

其维度变化是这样的：h * w * d * c -> Conv(2 * 2 * 2,s = 2) -> h / 2 * w / 2 * d / 2 * 2c

2.Decoder部分

2.1 Res-Block

我们将下采样提取出的特征，经过一个残差快得到残差特征，与上采样特征进行连接。

这个残差快里面是一个有两个”norm + 3 * 3 * 3 Conv”。最终输入与输出的维度不变。

其维度变化为：h * w * d * c -> Res-Block -> h * w * d * c。

2.2 Transpose Convolution（反卷积）

对链接特征进行反卷积，其分辨率提高为原来的2倍，特征通道减少为原来的一般。

其维度特变化为: h * w * d * c -> Trans-Conv -> 2h * 2w * 2d * 2c。

2.3 最后

输入的Patch-wise input经过残差快后的特征与上采样特征进行链接，经过一个带有softmax激活函数的1 * 1 * 1卷积来预测分割概率。

六 总结

作者自行总结了自己的贡献：

  • 我们 提 出 3D U X- N et 在 体积 设置 中纯 粹 用 Conv Net 模 块来 适 应 transf or me r 的 行为 。据 我 们所 知， 这是 第一 个利 用 3D 深 度卷 积在 体积 分割 任务 中与 transfor mer s ota 竞争 的大 型 内核块设计。
  • 利用 具有 L K 大 小 的 深度 卷 积作 为 通用 特征 提 取主 干， 并 引入 逐点 深 度卷 积 以以 较少 的 参 数有效缩放所提取的表示。
  • 我 们 使 用 三个 具 有 挑 战 性 的公共数据集在 1)直接训练和 2)使用体积多器官/组织分割的 微调 场 景 中评 估 3 D U X - Net。 3 D U X - N et 在 所有 Co nv N ets 和 transf or m ers S O T A 的两种 场景中以更少的 模型 参 数 实现 了 持续 的 改 进

3D UX-Net是第一个将分层 Transformer 的特征与用于医学图像分割的纯 ConvNet 模块相适应的网络结构哦。具体地，本文重新设计了具有深度卷积的编码器块，以更低的代价实现与 Transformer 想媲美的能力。最后，通过在三个具有挑战性的公共数据集上进行的广泛实验表明所提方法的高效性。
