MemoryAdaptNet：基于不变域级原型记忆的无监督领域自适应遥感语义分割

引言

语义分割是高分辨率遥感图像自编码的关键技术，并且在遥感领域备受关注。深度卷积网络由于其分层次的特征表达能力，已经成功应用于遥感分割任务。

然而，深度卷积网络的局限性：1.对大量标签标注数据的依赖；2.对数据分布差异的敏感，限制了卷积神经网络在遥感分割领域的应用。

本文提出了一种用于遥感语义分割的无监督自适应语义分割网络：MemoryAdaptNet。

I. INTRODUCTION

随着地球观测技术的快速发展，高分辨率遥感图像越来越多，这位地球观测提供了可能。高分辨率遥感图像的自译码在遥感分析领域扮演重要角色，语义分割是其重要工具。

基于深度卷积网络的语义分割模型存在以下问题：

（1） 对大量的数据标注具有依赖性。尽管用大量数据标注的模型是可行的，但是往往局限在固定的区域或应用上。对数据的标注耗时又耗力。

（2） 对数据分布差异特别敏感。由于地理、成像传感器、观察照明等成像条件的不同，高分辨率图像的外观和结构特点也不同，如下图所示。这种情况使得用特定得高分辨率标签数据集训练得深度模型在处理不同成像条件产生的图片时失效。

寻找一种能够准确有效处理域差异的遥感图像的语义分割模型至关重要。

无监督领域自适应可以将从有数据标签源域学的知识迁移到无数据标签的目标域中，弥合源域和目标域的差距。备受关注的对抗学习已经应用到了无监督自适应中，根据数据生成对抗损失的等级，对抗自适应方法可以分为三个等级：像素（pixel）级，特征（feature）级，输出（output）级。

虽然上述基于对抗学习的无监督自适应方法取得了一定成效，但他们大多数用于自然图像。鉴于自然图像和遥感图像有较大的差别，上面的方法不能直接用于遥感图像的语义分割。近年来，许多领域自适应语义分割方法用于遥感分割，大多数方法通过像素对齐或特征对齐来减少领域差异。现存的方法需要把大尺寸的遥感图像切成小尺寸的片（patch），然后成批地作为输入用于对抗训练。然而成批地源域或目标域的图像的数据分布并不能代表整个源域或目标域的数据分布，换句话说基于生成对抗损失的网络用成批图像来前向传播梯度，这导致提取的特征只能代表当前输入（该批次的源域或目标域的图片）的特征，并不能代表域不变特征，这种用当前输入提取的特征，我们称之为伪不变特征。理论上，伪不变特征和域不变特征之间存在一定的偏差和方差，这使得在源域上学到的知识无法最大程度的迁移到目标域上。我们的目标是，基于两个域的所有数据，通过领域自适应模型来表示域不变特征而不是伪不变特征。因此，如何从伪不变特征中提取域不变特征，对于交叉域的遥感语义分割至关重要。

为解决以上问题，本文提出了一种整合不变域级原型信息的无监督领域自适应遥感语义分割模型。鉴于output-leavel对抗自适应方法可以在输出空间保证特征的语义一致性，我们使用基于output-level的对抗机制来减小源域和目标域的差异并且提取具有语义一致性的伪不变特征。

如上图所示，我们用原型存储模型（prototype memory module）来存储历史源域或目标域图片的伪不变特征，同时，由于域不变特征和伪不变特征之间存在一定的偏差和方差，我们动态更新存储的伪不变特征来提取分类形式的域不变特征。我们用一个不变域级记忆聚合（IDMA）模型来自适应地合并原型存储模型中的域不变特征和当前的伪不变特征。当前的伪不变特征通过整合域不变特征得到了增强，使得模型可以最大化的将知识从源域迁移到到目标域，从而增强模型在目标域的表现。

本文的贡献主要为一下几点：

  1. 提出了不变域原型存储（invariant domain prototype memory）模型M来整合不变域级原型信息，M可以通过历史源域或目标域图片伪不变特征更新，提取不变域级原型信息。具体按类别的形式存储，以便对类别联合分布进行对齐。
  2. 引进了类别注意力导向IDMA（invariant domain-level memory aggregation）模型更好的将域不变特征整合到伪不变特征，增强当前输入图片的特征特征表示。
  3. 提出了基于熵的伪标签过滤机制以更好的更新模型M。特别的，保留熵小于等于阈值的预测值。

C. Memory Module

存储模型（memory module）使深度卷积网络可以存储超出局部小范围的变量和数据，被广泛应用于计算机视觉领域，如图像描述、运动物体检测、语义分割等。我们使用一种有效的特征记忆模块以分类的形式保留不变域级表示以保证分割网络获取比当前输入图片更多的域不变信息，增强在目标域图片的表现。

III. METHODOLOGY

我们的目标是，通过在源域中有标签的数据的学习，最终可以将模型学到的知识应用到目标域的无标签数据中。

如上图所示，我们提出的MemoryAdaptNet由领域特征对齐（DFA：domain feature alignment）模块和不变域级记忆聚合（IDMA：invariant domain-level memory aggregation）模块。DFA模块通过输出空间的对抗学习来调整源域和目标域图片间的特征。在IDMA中，我们用memory module来存储当前输入将DFA产生的伪不变特征，并且动态更新以获得域不变表示，将域不变表示整合当前的特征表示以细化最终的分类概率分布。当用目标域作为当前输入来更新域不变表示时，我们用基于熵的伪标签过滤机制预测目标域图片的伪标签，这有助于每个目标类别预测的计算。

A. Domain Feature Alignment

如上图所示，这是DFA模块的结构。DFA包含一个特征提取器F，分类器C1，判别器D， 其中F和C1构成了一个分割网络G1。F从源域和目标域的图片中提取特征，可以是任意的卷积网络，为了充分发挥高质量分割效果，我们选用在ImageNet上预训练的ResNet-101作为F的主干。该残差网络由33个残差块组成，每三个卷积层进行一次跳跃连接。在最后的残差快，我们用rate = 2的空洞卷积层（atrous convolution layer）来替代普通的3 * 3卷积层，为了提取更密集（更多）的特征映射和捕获更长范围的上下文。

B. Invariant Domain-Level Memory Aggregation

由特征提取器F提取的特征只代表当前输入批次图片的特征，并不能代表两个域的不变特征，是伪不变特征，这不能使从源域学到的知识最大程度的迁移到目标域。

memory模块可以是深度卷积网络的特征表达能力不知局限于当前的输入。我们使用calsswise memory模块将从DFA中获得的伪不变特征转换为域不变特征。具体的，就是把IDMA模块嵌入到DFA模块，

如上图所示，有一个不变域原型记忆模块（invariant domain prototype memory module）M，一个IDMA模块A，一个分类器C2，其中F、M、A、C2构成了分割网络G2。M用来存储从F得到的伪不变特征，并不断更新这些特征，在动态更新的过程中，M集成到所有源域和目标域的图片的伪不变特征，这是就可以近似域不变特征。具体来说，M分类的形式存储这些特征，以增强类内相似性和类间差异。

IDMA模块A将M中的不变域信息整合到当前输入图片的特征表示中来增强特征。然后我们用分类器C2来对增强的特征进行分类，以达到精确的分类预测。

A. Experimental Settings

  • 数据集：Vaihingen 2-D dataset and Potsdam 2-D dataset

  • 任务设定：

    • P2V_S task：Potsdam dataset with [IR, R, G]为源域，Vaihingen dataset with [IR, R, G]为目标域
